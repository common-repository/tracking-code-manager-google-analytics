== Description ==

= Wordpress Google Analytics Tracking Code Manager =

We believe that you shouldn't have to hire a developer to add Google Analytics to your website. That's why we built this Wordpress Google Analytics Tracking Code Manager plugin that's EASY and POWERFUL.

Wpbdcoders  allow you to connect your WordPress website with Google Analytics, so you can see how visitors find and use your website,

= Full Google Analytics Tracking Code Manager Feature List =

* Quick and Easy Setup - Easily setup Google Analytics for WordPress with just a few clicks (no coding needed)
* You just need to add tracking ID
* You can Specify the position of tracking code on front-end (Header, footer)
* You can disable tracking on specific page front page editor.

== Installation ==

1. Install Wordpress Google Analytics Tracking Code Manager  either via the WordPress.org plugin repository or by uploading the files to your server. (See instructions on <a href="https://codex.wordpress.org/Managing_Plugins">how to install a WordPress plugin</a>)
2. Activate Wordpress Google Analytics Tracking Code Manager.
3. Navigate to the Settings > Google Analytics Settings tab in your WordPress admin menu and configure the plugin.
